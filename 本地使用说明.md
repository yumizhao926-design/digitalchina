## 🔧 环境准备

### 1. 安装 Node.js

**Windows：**
1. 访问：https://nodejs.org/
2. 下载 LTS 版本（推荐 v20.x.x）
3. 运行安装程序，按提示完成安装
4. 打开 CMD，输入 `node --version` 验证

**Mac：**
```bash
# 使用 Homebrew 安装
brew install node

# 验证安装
node --version
```

**Linux (Ubuntu)：**
```bash
# 使用 NodeSource 仓库安装
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# 验证安装
node --version
```

### 2. 安装 pnpm（可选）

启动脚本会自动安装，也可以手动安装：

```bash
npm install -g pnpm
```

## ❓ 常见问题

### 问题1：端口被占用

**错误信息**：
```
Error: listen EADDRINUSE: address already in use :::5000
```

**解决方案**：

**Windows：**
```powershell
# 查找占用端口的进程
netstat -ano | findstr :5000

# 杀死进程
taskkill /PID <进程ID> /F
```

**Mac/Linux：**
```bash
# 查找占用端口的进程
lsof -i :5000

# 杀死进程
kill -9 <PID>
```

### 问题2：依赖安装失败

**解决方案**：
```bash
# 清理缓存
rm -rf node_modules package-lock.json pnpm-lock.yaml

# 重新安装
pnpm install
```

### 问题3：构建失败

**解决方案**：
```bash
# 清理构建缓存
rm -rf .next

# 重新构建
pnpm run build
```

### 问题4：页面显示空白

**解决方案**：
1. 检查控制台是否有错误信息
2. 清理浏览器缓存
3. 尝试使用无痕/隐私模式

### 问题5：无法访问 localhost

**解决方案**：
1. 检查服务是否启动
2. 检查防火墙设置
3. 尝试使用 `127.0.0.1:5000` 代替 `localhost:5000`

## 📊 性能优化

### 减少启动时间

1. **使用 SSD**：固态硬盘加载更快
2. **增加内存**：4GB+ 内存更流畅
3. **使用国内镜像**（如果 npm 下载慢）

```bash
# 配置 pnpm 使用淘宝镜像
pnpm config set registry https://registry.npmmirror.com
```

### 提升访问速度

1. 使用 Chrome 或 Edge 浏览器（性能更好）
2. 关闭不必要的浏览器扩展
3. 确保有足够的磁盘空间
